초등학교 3~6학년 학생이 집에서 스스로 학습할 수 있는 웹 기반 학습 게임 앱입니다.
로그인 없이 개별 프로필을 등록해서 사용하며, 수학·국어·영어 3과목과 구구단 게임을 5단계 난이도로 즐길 수 있습니다.

**🔗 바로 사용해보기: [https://windhak.github.io/study-app/](https://windhak.github.io/study-app/)**

## ✨ 주요 기능

- **프로필 관리**: 로그인 없이 학교/학년/반/번/이름/프로필명으로 프로필 등록, 여러 명이 함께 사용 가능
- **프로필 사진**: 프로필 생성 시 사진을 첨부하면 프로필 목록과 홈 화면에 표시, 언제든 사진 변경 가능 (기기 내 자동 리사이즈 후 저장)
- **자동 로그인**: 마지막으로 사용한 프로필로 자동 진입, 언제든 다른 프로필로 전환 가능
- **4가지 학습 콘텐츠**
  - ➕ **수학**: 덧셈·뺄셈·곱셈·나눗셈 (난이도 1~5단계, 자릿수와 연산 종류가 단계적으로 어려워짐)
  - 🔤 **국어 낱말게임**: 초성 힌트를 보고 낱말 맞히기
  - 📝 **국어 문장 연습**: 1~2단계는 문장 빈칸 채우기, 3~5단계는 생활 속 상황에 맞는 행동을 고르는 객관식 (친구 관계·안전 등)
  - 🔠 **영어 단어**: 영어 단어를 보고 4지선다로 뜻 맞히기, 🔊 버튼으로 발음 듣기 지원 (브라우저 내장 음성, 서버 불필요)
  - 🎮 **구구단 게임**: 2~9단 범위의 곱셈·나눗셈 문제
- **학습 설정**: 시작 전 "학습 시간(분)" 또는 "문제 수" 중 선택해서 진행
- **학습 기록**: 프로필별로 정답률, 문제 수, 스티커 보상 등을 기록하고 확인 가능
- **스티커 보상**: 세션 정답률 70% 이상 시 스티커 지급으로 학습 동기 부여
- **요정이 챗봇**: 규칙 기반(실제 AI 아님) 마스코트 챗봇으로, 인사·격려·과목 추천은 물론 가면라이더·파워레인저·로블록스 같은 아이들 관심사 잡담도 정해진 안전한 범위 안에서 가능

## 🛠️ 기술 스택

프레임워크 없는 **순수 HTML/CSS/JavaScript**로 제작되었습니다.

| 영역 | 선택 | 이유 |
|---|---|---|
| 마크업/스타일 | HTML5 + CSS3 (Flexbox) | 빌드 과정 없이 파일만 열어도 동작, 구형 브라우저 호환성 확보 |
| 로직 | Vanilla JavaScript | 구형 스마트폰 브라우저(저사양 WebView 등)에서도 안정적으로 동작 |
| 데이터 저장 | 브라우저 `localStorage` | 서버 없이 기기별로 프로필과 학습 기록 저장, 오프라인 사용 가능 |

구형 갤럭시 스마트폰처럼 오래된 기기의 브라우저에서도 문제없이 동작하도록 설계했습니다.

## 📁 프로젝트 구조

```
study-app/
├── index.html   # 화면 구조 (프로필 선택 · 홈 · 난이도 · 설정 · 퀴즈 · 결과 · 기록, 총 7개 화면)
├── style.css    # 스타일 (아이 친화적 컬러 시스템, 반응형 모바일 우선 레이아웃)
├── data.js      # 콘텐츠 데이터 뱅크(국어/영어 단어, 국어 문장·상황) + 문제 생성기(수학/구구단) + 요정이 챗봇 응답 데이터
└── app.js       # 화면 전환, 프로필/퀴즈/기록/챗봇 관리 로직
```

## 🚀 로컬에서 실행하기

빌드 과정이 필요 없습니다. 저장소를 내려받아 `index.html`을 브라우저로 열기만 하면 됩니다.

```bash
git clone https://github.com/windhak/study-app.git
cd study-app
# index.html을 브라우저로 열기
```

또는 간단한 로컬 서버로 실행할 수도 있습니다.

```bash
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

## 💾 데이터 저장 방식

별도의 서버나 데이터베이스 없이, 브라우저의 `localStorage`에 아래 정보를 기기별로 저장합니다.

- `studyapp_profiles`: 등록된 프로필 목록 (사진을 첨부한 경우, 리사이즈된 이미지가 `photo` 필드에 base64로 함께 저장됨)
- `studyapp_lastProfileId`: 마지막으로 사용한 프로필 (자동 진입용)
- `studyapp_records_{프로필ID}`: 프로필별 학습 세션 기록 (최근 100건)
- `studyapp_stickers_{프로필ID}`: 프로필별 누적 스티커 개수

⚠️ 데이터는 사용하는 기기와 브라우저에 저장되므로, 다른 기기에서는 학습 기록이 공유되지 않습니다.

## 🧚 요정이 챗봇에 대해

홈 화면의 "요정이와 대화하기"는 **실제 생성형 AI가 아니라, 미리 정해둔 키워드-응답 규칙으로만 동작하는 스크립트 챗봇**입니다.

- 서버나 외부 API 호출이 전혀 없고, 모든 응답은 앱 안에 내장된 문구 중에서만 선택됩니다.
- 대화 내용은 어디에도 전송/저장되지 않으며, 화면을 나가면 사라집니다.
- 인사·격려·과목 추천처럼 정해진 안전한 주제 범위 안에서만 반응하도록 설계했습니다.
- 가면라이더·파워레인저·로블록스 등 아이들이 좋아하는 콘텐츠에 대한 가벼운 잡담도 가능하지만, 실시간 인터넷 검색이 아니라 미리 조사해서 넣어둔 고정 정보이며 저작권 보호를 위해 일반적인 팬 대화 수준으로만 구성했습니다.
- 범위를 벗어난 입력에는 "선생님이나 부모님과 이야기해보라"는 식으로 안전하게 안내합니다.
